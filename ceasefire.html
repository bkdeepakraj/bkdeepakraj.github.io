Freelance: Ceasefire Violation, Dashboard for Ceasefire Incidents

UX Design - UI Design

Project Background

The Ceasefire Project is a comprehensive and methodologically rigorous open-access dataset on reported ceasefire incidents between India and Pakistan across the Line of Control (LoC) and International Border (IB) - in the form of an interactive dashboard.

Role

UX/UI Designer

Tools

Photoshop, Adobe XD, Illustrator

Date

2020

Target Audience and Problem the product solves

To create a interactive dashboard that helps to gain insights from the data on and around reported ceasefire incidents between India and Pakistan from 2004 onwards. The dashboard allows the users to explore the data with various filters and tools, enabling user to observe and analyse the phenomenon of ceasefire incidents between India and Pakistan.

The challenges the team wanted to solve with the project

To Create historical trends for reported data, to compare incidents types for both countries and to analyze with contextual factors with the available data

My role and people I worked in the project

My role was to create the interactive dashbaord with map view

Interacted with Client for requirement gathering and Developers to understand the Technical Constraints. Refined the requirement as the project progressed to align UX/UI with the requirements
