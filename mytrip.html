Holiday Trip Booking App.

Project Concept Name

My Holiday Trip Planner

Project Summary

MyTripPlanner app provides customers to Book an awesome holiday plan for a travel enthusiast. The idea is to provide a service where user can search for holiday destinations based on preference, customize the holiday plan, book travel itinerary, book accommodation in hotels or rent home, see reviews of other users. By Customers booking travel itinerary and booking accommodation MyTripPlanner wants to generate revenue by partnering with travel and hotel industry.

Assuming that MyTripPlanner has built an algorithm to suggest users destination places based on preferences and also create holiday packages by providing best affordable price by comparing other sites using search bots and the algorithm changes the destination places using anticipatory feedback based on the search term, preferences, age, visited places and etc.

Note on the Business Requirement: Assuming Many of the users have some kind of mental model how booking holiday trip sites or app works.

Below, I have posted the figma prototype which shows the design of various UX Flows like Onboarding Flow, Search, Create Pacakge, Book or Plan a trip. The UX flows have been simulated for demo purpose with some pre-filled data.

I used Design thinking as UX Process to build the product

  1. Understand - Problem Statement. I picked three problem statements that need to be solved from business prespective and satisfy the user needs by understanding the purpose of Holiday Booking
  2. Understand - Persona. I considered three Persona target types. Genz(Book or Plan Trip), Millenials(Book or Plan Trip) and Baby Boomers(Create package) as users for the App. I worked on the UX Flows based on the persona types.
  3. Observe - User Journey Map. The whole user journey map was created based on the Persona types and Registered/Non-Registerd Users.
  4. Define - How Might We?
    • How might we help users to find the right destination?
    • How Might we help the users search for destination with different search critieria?
    • How Might we help display the information to the customers based on thier needs?
    • How might we help users to create pacakge based on preferences?
    • How might we help users to book or plan a trip with ease and complete booking?
  5. Ideation - Wireframe. I created wireframes to test the concept. Built Low fiedlity wireframes using figma to get the initial idea to perfection. During the process i followed the interaction design principles.
  6. Prototype - Built prototype with actual visual designs to shape the final product.
