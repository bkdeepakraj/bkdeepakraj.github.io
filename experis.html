Experis Responsive Website

Interaction Design - Visual Design - Responsive Design Principles

Project Summary

Experis.com, A site for searching white collar Jobs. Users can create profile and search for jobs. User can search for jobs, create profile and set preferences for the job. User can use advanced search based on their location preferences and also locate branch near surroundings.

The UX Challenge

The project was to redesign to responsive site. The challenge was to re-design the existing screens to responsive with the same set of initial screens that were developed.

The UX Approach

Through design iteration the screens were re-designed to responsive design. To make it responsive, I collaborated with Bootstrap developers and the site was revamped into responsive site.

Stakeholders

Art director from Manpower(client) and Wipro Business Analyst

Visual Design

The client had created style guide/Design System for the site. The UI was designed to match the style guide before making it responsive.

Responsive site

The website was designed responsive and coded with Bootstrap
