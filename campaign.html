Campaign Studio, Desktop Application

Project Summary

Campaign Studio is a desktop application for Campaign managers(users) who run Marketing campaigns and offers in PayPal website. Based on user profiling and rule based logical conditions the offers will be displayed to the user in the PayPal website.

The UX Challenge

The challenge was to design the experience for campaign managers to create a rule based logical conditions to display marketing and offers to users who use PayPal.com.

The UX Approach

Through observation of the old system and analyzing the data the campaign managers use, I created a UI where users can create logical rule (and & or) conditions to display the marketing messages and offers to the users. And the user can see the status of the campaigns and offers they have created.

Stakeholders

Campaign Managers and PayPal Business Analyst.

Interaction and Visual Design

Created Task flow, Information Architecture and Wireframes. Visual Design was created based on PayPal style guide.

Task flow, Search Campaigns, Create Rule UI and Campaign details screens.
